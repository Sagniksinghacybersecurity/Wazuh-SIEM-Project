# 🔐 Wazuh SIEM Project - File Integrity Monitoring

## 📌 Overview
This project demonstrates a real-time File Integrity Monitoring system using Wazuh SIEM.

## 🧠 Objective
- Implement SIEM using Wazuh
- Monitor file changes
- Detect real-time alerts

## ⚙️ Tools Used
- VMware Workstation
- Ubuntu (Wazuh Server)
- Windows (Agent)
- Wazuh SIEM

## 🏗️ Architecture
Windows Agent → Wazuh Manager → Wazuh Indexer → Dashboard

## 🔧 Features
- Real-time file monitoring
- Alert generation
- Threat detection

## 📊 Results
- File creation detected
- File modification detected
- File deletion detected
